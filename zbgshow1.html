zbgshow1 とは

Ｚ壁紙ディスプレー１号のことで、壁紙ごとに HTML ファイルを作ることなく、壁紙の背景サンプル表示を行う CGI です。壁紙の背景サンプルは、別ウインドウ上に表示します。

動作条件

  • ホームページがあるサーバで（Perl5）以上を使用したCGIが使えること。
  • サーバのOSがLinux、UNIXで、文字コードは EUC_JP もしくは SJIS であること。たぶんUTF8ではだめです。古いんで・・
  • サーバのOSがLinux、UNIXで、Web Server が Apache で動作確認しています。

ファイル構成 と アクセス権

  • zbgshow1.cgi - rwxr-xr-x [ 755 ]　--- Ｚ壁紙ディスプレー１号CGI本体プログラム
  • このプログラムファイル一本です

ユーザー変数の編集 [ zbgshow1.cgi ]

  • 設定項目は以下の2項目です。
    1. 壁紙ディスプレーページのタイトル [ $Title ]
    2. 壁紙ディスプレーページに表示する簡単な文章 [ $Body ] 

              ==== zbgshow1.cgi ====
              ..
              ..
              ###################################################################
              #
              #   カスタマイズ可能変数（お好みで書き換え可）
              #
              ###################################################################
              
              $Title = '壁紙サンプル表示';
              $Body  = '表示はこんな感じです。<br>いかがですか？';
              ..
              ..
  • 壁紙ディスプレーページのタイトル [ $Title ] は、壁紙が背景となったページに表示するタイトルです。
  • 壁紙ディスプレーページに表示する簡単な文章 [ $Body ]は、壁紙が背景となったページに表示する文章です。
  • [ $Body ] は、HTML タグが使用 OK です（「HTMLで定義してください」と言うべきか、、）。

設置例とパラメータ

<a href="/home/cgi-bin/zbgshow1.cgi?url=/home/img/kabegami.gif&title_col=FF0000&body_col=0000FF"
　　target="zbgshow1"&>
<img src="/home/img/kabegami.gif">
</a>
  • 'url' パラメータで、背景画像の URL を指定します。
  • 'title_col' パラメータで、タイトル文字色を指定します。
  • 'body_col' パラメータは、本文の文字色を指定します。
