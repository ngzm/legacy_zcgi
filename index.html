Legacy zcgi

View project on GitHub

2001年くらいに作成した古い Perl CGI スクリプトです。 古いままです。
ほとんどのCGIは jcode.pl の関係により、サーバの文字コードは EUC か SJIS しか動作しないと思います。

Zcounter1

Zcounter1 の説明はこちら
  • Zcounter1 とは Ｚカウンター１号のことです
  • Ｚカウンター１号は、GIFイメージのアクセスカウンターです
  • 同一IPアドレスのチェック機能（オプション）とかがあります

Zounter2

Zcounter2 の説明はこちら
  • Zcounter2 とは Ｚカウンター２号のことです
  • Ｚカウンター２号は、GIFイメージのアクセスカウンターです
  • 同一IPアドレスのチェック機能（オプション）あります
  • アクセス管理機能も付いてます
  • アクセス管理機能は、アクセス日時やアクセス元情報が収集できます
  • さらにアクセスログのダウンロードや削除ができます

Zguest1

Zguest1 の説明はこちらへ
  • Zguest1 は Ｚゲストブック１号機と呼ぶやつです( ^^)
  • 昔ながらのアイコン付きの楽しげなゲストブックです
  • 新規投稿にはだれでもレスが付けられます
  • クッキー対応で２回目以降の書込みが楽っす
  • 管理機能付き、投稿ログの編集、削除、ダウンロードがきます

Zbgshow1

Zbgshow1 の説明はこちらへ
  • Ｚ壁紙ディスプレー１号機と呼んでます(´・ω・`)
  • Ｚ壁紙ディスプレー１号は、壁紙の背景サンプル表示を行う CGI です
  • 壁紙の背景サンプルは、別のウインドウ、或いは、別のフレーム上に表示します

Zbgshow12

Zbgshow2の説明はこちらへ
  • こちらは、Ｚ壁紙ディスプレー２号となります
  • Ｚ壁紙ディスプレー２号は、壁紙のサンプル表示を行う CGI です
  • Ｚ壁紙ディスプレー２号は、別ウインドウやフレームを開くことなく壁紙をサンプル表示します
