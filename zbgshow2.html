zbgshow2 とは

Ｚ壁紙ディスプレー２号のことです。これは、壁紙ごとに HTML ファイルを作ることなく、壁紙の背景サンプル表示を行う CGI です。現在のウインドウに、選択した壁紙の背景を表示ができます。

動作条件

  • ホームページがあるサーバで（Perl5）以上を使用したCGIが使えること。
  • サーバのOSがLinux、UNIXで、文字コードは EUC_JP もしくは SJIS であること。たぶんUTF8ではだめです。古いんで・・
  • サーバのOSがLinux、UNIXで、Web Server が Apache で動作確認しています。

ファイル構成 と アクセス権

  • zbgshow2.cgi - rwxr-xr-x [755] --- Ｚ壁紙ディスプレー２号CGI本体プログラム
  • このプログラムファイル一本です。

設置例とパラメータ

壁紙素材ページ(HTML)から Ｚ壁紙ディスプレー２号をリンクする記述例です。以下の例を参考にHTML ファイルを記述してください。
なお、赤で表示しているところは、皆さんの環境に従って変更すべき箇所です。

<a href="/home/cgi-bin/zbgshow2.cgi/home/html/kabegami.html?url=/home/img/kabegami.gif">
<img src="/home/img/kabegami.gif">
</a>
  • '/home/cgi-bin/zbgshow2.cgi' は、Ｚ壁紙ディスプレー２号 CGI の URL です。環境にあわせて変更してください。
  • 次の '/home/html/kabegami.html' は、現在ページの URL です。この部分も環境にあわせて変更してください。
  • 続く 'url' パラメータ：この例では'/home/img/kabegami.gif' の部分で、背景画像の URL を指定します。
  • 現在ページの URL や背景画像のURLは、CGIから参照できる必要がありますので、この例のように絶対パスで指定してください。
